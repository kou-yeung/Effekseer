Fカーブ

Overview

It is a function that you can set the value for each frame by using graph.

How to use

Display graph

By specifying the F curve with each item (position, rotation, etc.), the graph is displayed in the F curve window.

Movement / scaling of graphs

Move graphOn the graph, hold down the mouse wheel and move the mouse
Horizontal / scalingOn the graph, hold down Ctrl and rotate mouse wheel
Vertical / scalingOn the graph, hold down Shift and rotate mouse wheel

Add key frame

In the case of a three-dimensional vector

XOn the graph, press the X key
YOn the graph, press the Y key
ZOn the graph, press the Z key

In the case of a color

RedOn the graph, press the R key
GreenOn the graph, press the G key
BlueOn the graph, press the B key
AlphaOn the graph, press the A key

Select key frame

You can select by left clicking the mouse.

Move key frame

After selecting the key, you can move by moving the mouse while holding down the left mouse button. In the same way, you can also move the keys for controlling the curve.

Delete key frame

After selecting the key, you can delete it by pressing Delete.

Parameter

フレーム

Set for each key. Specify the frame of the key.

値

Set for each key. Specify the value of the key.

補完

Set for each key. Specify the completion method between the key and the key.

先頭

Set it for each graph. Specify how to set the value of the graph before the left edge of the graph.

末尾

Set it for each graph. Specify how to set the value of the graph after the right edge of the graph.

Sampling

Set it for each graph. When actually playing back, specify how many frames to use to display. The smaller the memory is used, the closer the shape of the graph is used.

左x,y

Set for each key. Specify the position of the left handle of the curve control of the key.

右x,y

Set for each key. Specify the position of the right handle of the curve control of the key.

オフセット最大,最小

Set it for each graph. When using values, offset the range, randomly shift the value up or down the graph.
