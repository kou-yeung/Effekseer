Position

Overview

Among the parameters related to the position of the node, set the parameters concerning the position which is particularly affected by the parent particle.

Parameter

Position

Specify the position of the particle. Here we specify the position relative to the parent particle. With Basic settings, if you do not take influence from parents, specify absolute position.

Set Position

Place the particles in a fixed position.

PVA

Specify the position, velocity and acceleration of the particle.

Easing

Specify the position at the time of particle generation and the position at the end.

F-Curve

Use F-Curve.
