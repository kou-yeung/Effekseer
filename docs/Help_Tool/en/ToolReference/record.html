録画

Overview

It outputs the created effects as serial number images, images and gif animation. This function is very useful in situations such as when posting effects created on Twitter or Blog etc., when you can not directly play effects created with Effekseer, or when loading the effects playback is concerned.

パラメーター

録画範囲(解像度)

横幅縦幅

Sets the size of the image per animation.

録画範囲を表示

When checked, the recording range is displayed on the screen.

出力フレーム

開始フレーム

ファイルに出力する映像の範囲の開始フレームを設定します。

終了フレーム

Sets the start frame of the range of the image output to the file.

出力頻度(フレーム毎)

Sets the frequency of file output. For example, if 1 is set, 60 frames are recorded, 60 images will be output. When 2 is set, when 60 frames are recorded, 30 images are output. If you increase the number like this, the file size will be smaller and the image will be skipped instead.

出力形式

There are 3 types of "1枚にまとめて出力" "連番画像として出力" and "Gifアニメーションとして出力".

"1枚にまとめて出力" is an image with animation arranged from the upper left to the upper right. Images are arranged horizontally by the number specified in the horizontal direction number. Images in the number of recorded frames / horizontal direction are arranged in the vertical direction.

"連番画像として出力" outputs numbers with numbers attached to shot images.

"Gifアニメーションとして出力" will output the captured image as Gif animation.

オプション

背景色を透明にする

If not checked, the background is black, but when you check it the background will be transparent and will be output. However, it is invalid for gif animation.
