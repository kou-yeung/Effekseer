Render - Model

Overview

Drawing and parameters when "Model" is selected in drawing setting are explained.

When "Model" is selected, the 3D model read from the outside is displayed.

The color / distortion image common to the drawing is used for the image related to the color used for the model. Depending on the model, display may be incorrect unless depth writing and depth test are enabled.

パラメーター

モデル

Specify a model file for Effekseer (.efkmodel).

法線画像

Specify the normal image to be used for the 3D model.

ライティング

Specifies whether the model is affected by the light source.

カリング

Specify the direction of the face to be displayed by the model.

全体色

Specify the color tone of the whole model.
