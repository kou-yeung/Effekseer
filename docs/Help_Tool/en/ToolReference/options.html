オプション

Overview

Set options related to editor display such as grid and background.

パラメーター

Grid Color

Specify the color of the grid.

Grid Visibility

Specify whether to display the grid or not.

X-Y Grid Visibility

Specify whether to display grid in XY plane.

X-Z Grid Visibility

Specify whether to display grid on XZ plane.

Y-Z Grid Visibility

Specify whether to display grid on YZ plane.

Grid Size

Specify the size of the grid.

Light Direction

Specify the direction of the light that illuminates the model when displaying the model.

Light Color

Specify the color of the light.

Ambient Color

Specify the color of ambient light.

Output Magnification

This is a parameter that reproduces the enlargement when changing the magnification ratio when outputting the effect file to the outside. When this parameter is set, the effect is enlarged and displayed.

Output FPS

It is a parameter that reproduces the display when the effect is reproduced by changing the FPS externally. When this parameter is set, FPS is changed and displayed.

Coordinate System

It is a parameter that reproduces the display when the effect is reproduced by changing the coordinate system externally. When this parameter is set, the coordinate system is changed and displayed.

Background Color

Specify the color of the background.

Background image

Specify the image of the background.

Color Space

It is a parameter that reproduces the display when changing the color space and playing the effect externally. There are gamma space and linear space in the color space.

Gamma space Linear space

Unity and others, recent game engines can change the color space. Set according to the color space to use. Please check the description of each game engine etc. for the meaning of the color space itself. Effekseer's color space is gamma space by default.

マウスの回転方向(X)

Specify the rotation direction of the mouse.

マウスの回転方向(Y)

Specify the rotation direction of the mouse.

マウスのスライド方向(X)

Specify the slide direction of the mouse.

マウスのスライド方向(Y)

Specify the slide direction of the mouse.
