ネットワーク

Overview

Set parameters related to the network. You can edit the effect of an application (game etc.) incorporating the running runtime via the network from the outside. If you would like to find out more, please read the item about the runtime network.

パラメーター

接続先

Write the IP address or DNS of the hardware running the application.

ポート

Enter the port number of the server running in the application.

切断時、自動接続を一定時間ごとに行う

If it is not connected anywhere, it tries to connect to the connection destination set at a fixed time.

ファイル読み込み時にデータを送信する

If the connection is established when reading the file, send the effect data to the application.

編集時にデータを送信する

If the connection is established when editing the effect, send effect data to the application.

保存時時にデータを送信する

If the connection is established when saving the file, send effect data to the application.

データ送信

Forcefully send effect data to the application.

接続

Connect to the specified connection destination.
